# Qlik Sense APIs (qlik-sense)
APIs for Qlik Sense, a business intelligence and data analytics platform

**URL:** [Visit APIs.json URL](https://qlik.dev/apis.json)

## APIs

### Qlik Sense Engine API

**Human URL:** [https://qlik.dev/apis/json-rpc/qix](https://qlik.dev/apis/json-rpc/qix)


#### Tags:

 - Analytics, Business Intelligence, Data Engine, WebSocket

#### Properties
